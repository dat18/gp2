# 後期10回目(12/12)
## Gitのアップデート
以下、ご協力お願いします。

### Git for Windows
- https://gitforwindows.org/ を開く
- Download したら、クリックして実行
- 詳細をクリックして、実行
- パスワードが必要になったら呼んでもらう
- 設定はすべてデフォルトのままで進めて、インストールを完了させる

### GitHub Desktop
- GitHubデスクトップを起動
- Helpメニューから、About GitHub Desktopをクリックして開く
- ダウンロードが始まっていたら、完了するのを待つ
- ダウンロードが完了していて、ボタンが表示されていたらクリックすると、GitHub Desktopが閉じて、アップデートが始まる

### VisualStudio
- スタートメニューから、Visual Studio Installerを起動
- Visual Studio 2019の更新をクリック。その前に、インストーラーの更新が必要といわれた場合は、更新ボタンをクリックして、更新
- 更新を実行

## ブログの書き方
- 週1なので短くてよい
  - 参考：[コガネブログ](http://baba-s.hatenablog.com/)
- 書く内容
  - ブログの概要を1～数行でざっくりと。スクリーンショットがあるとよい
  - 目次
  - 前提条件として、Unityのバージョンや、アセットを使っていたらそのバージョン
  - 本文
    - 簡潔に書く。省略して意味が変わらない表現があれば、短い方を採用する
    - スクリーンショット多めがよい
    - 技術ブログはくだけた表現が許されるが、企業に見せても構わないようにすること(就活で使えなければ意味がないので)
  - まとめで、ブログの内容について簡単におさらい
  - 最後に利用したアセットや、参考にしたブログ、動画などのURLを**参考URL**などでまとめておく

# 7回目(6/6)
## 予定
- Unity2019とVisual Studio2019のインストール

## 手順
### 準備
1. 担当PCの割り振り
  - 一度にすべてをやるとネットで遅くなるので、半分をインストールしつつ、残りのPCでは作業を進められるようにしておく
1. PCをStudentで起動
1. `a館3f-6f共用`にある`A601Install`フォルダーを、デスクトップにコピーする

### Unity Hubのインストール
Unity Hubがインストールされていたら、Unity2019のインストールに進む。

- デスクトップにコピーしたフォルダーの中の`UnityHubSetup`をダブルクリック
- パスワード入力が出たら教員を呼んで入力してもらう
- デフォルトの設定でそのままインストールして、起動
- 起動したら右上のアカウントアイコンをクリックしてサインインをクリック
- *Email*に`unity@dat.ac.jp`と入力して、*Password*欄をクリックしてカーソルを移動させたら、教員を呼ぶ
- *新規ライセンスの認証*をクリック
- *Unity Personal*をクリック > *Unityを商用に利用しません*をクリック > *実行*をクリック
  - 下にいろいろ表示されるが無視してよい
- 左上の←をクリック

Unity2019のインストールの3.へ。

### Unity2019のインストール
1. Unity Hubを起動
1. Unity Hubを更新
  - パスワードを求められたら呼んでください
1. 左にある*インストール*をクリック
1. 右上のインストールをクリック
1. *Unity2019.1.5f1*をクリックして、*次へ*をクリック
1. 以下のモジュールを追加
  - WebGL Build Support
  - Windows Build Support(IL2CPP)
  - 上記以外にチェックがあったら外す
1. 実行をクリック
1. 教員を読んで、パスワード入力

インストールに時間がかかるので、他のPCのインストールを始める。完了したら、Visual Studio 2019のインストールへ。

### Visual Studio 2019のインストール
1. デスクトップにコピーした`A601Install`フォルダーにある`vs_community__170873077.1559792595`をダブルクリック
1. 教員にパスワードを入力してもらう
1. [こちら](https://ameblo.jp/minato58-work/entry-12451856226.html)を参照して、Visual StudioのインストールとUnityの設定を完了させる

### Visual Studio 2019の起動
- サインインは教員を呼ぶ
- 開発設定を*Visual C#*、色は*濃色*で*Visual Studioの開始*

### 動作確認
1. Unity Hubで新規プロジェクトを作成。名前は何でもよい
1. スクリプトを作成して、ダブルクリックして、Visual Studioが正常に起動するか確認する

以上、確認ができたら作成したプロジェクトを削除して、インストール完了。

### アンインストール
1. スタートメニューをクリックして、歯車の設定アイコンをクリック
1. *アプリ*をクリックして、一覧が表示されるまでしばらく待つ
1. *Visual Studio Community 2017*をクリックして、*アンインストール*をクリック
1. 同様に、Unityをクリックして、`2017.X.X`であることを確認したら、アンインストールする

# 6回目(5/30)
- 仕上げ
- ゲーム動画の撮影
  - 準備
    - スタートメニューから歯車アイコンをクリックして、ゲームをクリック
    - 一番上の設定をオンにする
    - 設定ウィンドウを閉じる
  - 録画
    - 録画する画面を表示したら、[Win]+[Alt]+[R]キーを押す
    - 「ゲーム機能は利用できません」というウィンドウが出たら、チェックの場所をクリックする
    - ゲーム画面をクリックする
    - 録画中は画面に操作バーが表示されている(これは録画されない)。停止したい場合は停止ボタンを押す
  - 録画を確認する
    - [Win]+[G]キーを押す
    - 「全キャプチャを表示」をクリック
    - 「ファイルの場所を開く」をクリックしたら、動画ファイルの保存先フォルダーを確認できる
